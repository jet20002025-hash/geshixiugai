# 快速配置清单 - geshixiugai.org

## ✅ 配置步骤（按顺序执行）

### 第一步：在 Cloudflare 添加域名

1. 访问 https://dash.cloudflare.com
2. 点击 **"Add a Site"**
3. 输入：`geshixiugai.org`
4. 选择 **Free Plan**（免费计划）
5. 点击 **Continue**
6. Cloudflare 会显示两个 Nameservers，**复制保存**：
   ```
   [示例，实际会不同]
   dante.ns.cloudflare.com
   lola.ns.cloudflare.com
   ```

### 第二步：在 Spaceship 修改 Nameservers

1. 登录 Spaceship 账户
2. 找到域名 `geshixiugai.org`
3. 进入 **DNS 管理** 或 **Nameservers** 设置
4. 将 Nameservers 改为 Cloudflare 提供的两个地址
5. 保存更改
6. ⏰ 等待生效（通常 5 分钟到几小时）

### 第三步：在 Cloudflare 配置 DNS 记录

1. 在 Cloudflare Dashboard 中，选择域名 `geshixiugai.org`
2. 点击左侧 **DNS** → **Records**
3. 删除所有现有记录（如果有）
4. 添加以下记录：

   **记录 1:**
   - 类型：`CNAME`
   - 名称：`@`
   - 目标：`cname.vercel-dns.com`
   - 代理状态：✅ **已代理**（橙色云朵）
   - TTL：`Auto`

   **记录 2:**
   - 类型：`CNAME`
   - 名称：`www`
   - 目标：`cname.vercel-dns.com`
   - 代理状态：✅ **已代理**（橙色云朵）
   - TTL：`Auto`

5. 点击 **Save**

### 第四步：在 Vercel 添加自定义域名

1. 登录 Vercel Dashboard
2. 进入你的项目
3. 点击 **Settings** → **Domains**
4. 点击 **Add Domain**
5. 输入：`geshixiugai.org`
6. 点击 **Add**
7. Vercel 会自动验证域名（可能需要几分钟）

### 第五步：等待生效

- ⏰ DNS 传播：5 分钟到 24 小时
- ✅ SSL 证书：自动配置（通常几分钟）
- 🔍 验证：访问 https://geshixiugai.org

## ⚠️ 常见问题

### 问题1: Nameservers 在哪里修改？

**Spaceship 操作步骤：**
1. 登录 Spaceship
2. 进入域名管理
3. 找到 "Nameservers" 或 "DNS" 设置
4. 修改为 Cloudflare 提供的 Nameservers

### 问题2: 找不到 CNAME 记录选项？

**解决方案：**
- 确保在 Cloudflare DNS 设置中
- 点击 "Add record"
- 选择类型为 "CNAME"
- 名称填写 `@`（代表根域名）

### 问题3: 访问显示 "Not Found"？

**可能原因：**
1. DNS 还未生效（等待更长时间）
2. Vercel 域名未正确添加
3. DNS 记录配置错误

**解决方法：**
1. 检查 Vercel 项目设置中的域名状态
2. 确认 DNS 记录指向 `cname.vercel-dns.com`
3. 等待 DNS 传播完成

### 问题4: SSL 证书未生效？

**解决方法：**
1. 在 Cloudflare 中，确保 SSL/TLS 设置为 "Full" 或 "Full (strict)"
2. 在 Vercel 中，等待自动 SSL 配置（通常几分钟）
3. 清除浏览器缓存后重试
