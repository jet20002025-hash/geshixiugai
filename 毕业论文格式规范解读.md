# 毕业论文格式规范解读

## 📋 文档概述

这是一份完整的**毕业设计（论文）写作规范和参考格式**文档，包含：
1. **写作规范**：详细的格式要求和写作标准
2. **参考格式**：具体的排版格式要求
3. **示例论文**：完整的激光打靶系统毕业设计论文

---

## 📑 文档结构分析

### 第一部分：写作规范

#### 1. 论文的八个主要组成部分

按顺序包括：
1. **题目** - 不超过20个汉字，可加副标题
2. **中、英文摘要** - 中文不超过400字，英文与中文对应
3. **关键词** - 3-6个，中英文对应
4. **目录** - 三级标题，标明页数
5. **文本主体** - 引言、正文、结论
6. **致谢** - 简短致谢
7. **参考文献** - 不少于10篇，外文不少于2篇，期刊论文不少于4篇
8. **附录** - 必要时添加

#### 2. 标题序次结构

两种序次方式：
- **第一种**（文科类）：一、（一）、1……
- **第二种**（理工科类）：1.、1.1、1.1.1……

#### 3. 文本主体要求

- **引言**：说明课题意义、目的、研究内容、范围
- **正文**：详细表述、深入分析、周密论证
- **结论**：概括工作成果，指出优缺点，提出改进方向

#### 4. 格式细节要求

**（1）标题**
- 小标题使层次清晰
- 可用文字概括或直接使用数字

**（2）注释**
- 页末注或篇末注
- 序号用①、②、③（不是[1]、[2]）

**（3）标点符号**
- 半字线"-"：用于并列关系（温度-时间曲线）
- 一字线"—"：用于化学键、标准代号等
- 省略号：正文中占两格"……"，公式中占一格
- 括号：圆括号为主，双重括号用圆括号+方括号

**（4）名词、名称**
- 使用标准科技名词
- 外文缩写首次出现需注明含义
- 外国人名：知名人士用译名，其他用英文原名

**（5）量和单位**
- 使用我国法定计量单位
- 单位名称全称、简称要一致
- 非物理量单位可用汉字+符号

**（6）数字**
- 测量、统计数据用阿拉伯数字
- 公历年月日用阿拉伯数字
- 普通叙述中不很大的数目用汉字

**（7）公式**
- 另起一行书写
- 编号用圆括号放在公式右边
- 引用时写成"由式（16.20）可见"

**（8）表格**
- 有表题和表序
- 表题在表格上方正中
- 允许下页接写，需重复表头

**（9）插图**
- 有图序和图题
- 图题在图的下方
- 分图用a、b、c标序

#### 5. 参考文献格式

按GB7714-1987标准，包括：
- 期刊：[序号] 作者．文献题名[J]．刊名，出版年份，卷号(期号)：起止页码
- 专著：[序号] 著者．书名[M]．出版地：出版者，出版年：起止页码
- 论文集、学位论文、报告、专利、标准、报纸、电子文献等

### 第二部分：参考格式

#### 1. 页面设置

- **纸张**：A4纸
- **页边距**：
  - 上：3厘米
  - 下：2厘米
  - 左：3厘米
  - 右：2厘米
  - 装订线：1厘米
- **页眉距边界**：2厘米
- **页脚距边界**：1厘米

#### 2. 页眉页脚

- **页眉**：除封面、封底外，均采用"杭州电子科技大学本科毕业论文"或"杭州电子科技大学本科毕业设计"
- **字体**：宋体五号字，居中
- **页码**：封面、封底、中文摘要、ABSTRACT、目录无页码；其余部分用阿拉伯数字，Times New Roman五号字，居中

#### 3. 字体字号要求

| 元素 | 字体字号 | 格式要求 |
|------|---------|---------|
| "摘要"、"目录"等标题 | 黑体三号字 | 居中，单倍行距，段前2行，段后2行 |
| "ABSTRACT" | Times New Roman加黑三号字 | 居中，单倍行距，段前2行，段后2行 |
| 中文摘要内容 | 宋体小四号字 | 行距20磅 |
| 英文摘要内容 | Times New Roman小四号字 | 行距20磅 |
| "关键词" | 黑体小四号字 | 行距20磅 |
| "Keywords" | Times New Roman加黑小四号字 | 行距20磅 |
| 关键词内容 | 宋体小四号字（中文）/Times New Roman小四号字（英文） | 行距20磅，用分号分开 |
| 目录内容 | 宋体小四号字（中文）/Times New Roman小四号字（英文） | 至少列出第一和第二级标题 |
| 正文第一级标题 | 黑体三号字 | 居中，单倍行距，段前2行，段后2行 |
| 正文第二级标题 | 黑体四号字 | 靠左空两格或顶格（根据序次方式） |
| 正文第三级标题 | 黑体小四号字 | 靠左空两格或顶格（根据序次方式） |
| 正文内容 | 宋体小四号字（中文）/Times New Roman小四号字（英文） | 行距20磅，标准字符间距 |
| 图名、图号 | 宋体五号字 | 居中，列在图的下方 |
| 表名、表号 | 宋体五号字 | 居中，列在表的上方 |
| 公式 | 宋体五号字 | 另起一行，居中 |
| 参考文献 | 宋体小四号字（中文）/Times New Roman小四号字（英文） | 居左顶格，行距20磅 |

#### 4. 其他格式要求

- 每一章内容均另起一页
- 目录可使用Word系统自带的"插入目录功能"自动生成
- 封面由学校统一制作，填写内容为楷体小三号字

## 🎯 关键要点总结

### 1. 结构完整性
- 必须包含八个主要部分
- 顺序不能颠倒
- 每个部分都有具体要求

### 2. 格式规范性
- 严格按照字体、字号、行距要求
- 页眉页脚格式统一
- 标题层次清晰

### 3. 内容质量要求
- 参考文献不少于10篇
- 外文文献不少于2篇
- 期刊论文不少于4篇
- 注释和参考文献要区分清楚

### 4. 排版细节
- 标点符号使用规范
- 数字使用规范
- 公式、表格、插图格式规范
- 量和单位使用规范

## 📌 注意事项

1. **不同学校可能有不同要求**，这份文档是杭州电子科技大学的标准
2. **文科和理工科的标题序次不同**，需要区分
3. **注释和参考文献的序号格式不同**，不能混淆
4. **中英文格式要求不同**，需要分别处理
5. **公式、表格、插图的格式有特殊要求**，需要特别注意
