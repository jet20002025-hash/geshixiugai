## 第一步：进入备案系统

### 1.1 访问备案系统

1. **访问备案系统**
   - 直接访问：https://beian.aliyun.com/
   - 或登录阿里云控制台 → 点击顶部"**备案**" → "**ICP备案**"

2. **登录账号**
   - 使用购买服务器和域名的阿里云账号登录

### 1.2 开始备案

1. **点击"开始备案"**
   - 在备案系统首页，点击"**开始备案**"按钮

2. **选择备案类型**
   - 选择"**首次备案**"（如果是第一次备案）
   - 点击"**下一步**"

---

## 第二步：填写备案信息

### 2.1 填写主体信息（个人备案）

**重要：所有信息必须真实有效！**

1. **证件类型**
   - 选择"**身份证**"

2. **证件号码**
   - 填写18位身份证号码

3. **姓名**
   - 填写身份证上的真实姓名

4. **手机号码**
   - 填写手机号（需要接收验证码）
   - 点击"**获取验证码**"
   - 输入收到的验证码

5. **邮箱**
   - 填写常用邮箱（用于接收备案通知）

6. **通信地址**
   - 填写详细地址（省市区+详细地址）
   - 例如：浙江省杭州市西湖区xxx街道xxx号

7. **点击"下一步"**

### 2.2 填写网站信息

**重要：使用新域名 geshixiugai.cn**

1. **网站名称**
   - 填写：**诚叔工具站**
   - 或：**格式修改工具站**
   - **注意**：不能包含"中国"、"国家"等敏感词

2. **网站域名**
   - 填写：**geshixiugai.cn**
   - **注意**：不要带 www 或 http://
   - **重要**：使用新域名，不是 geshixiugai.org

3. **网站服务内容**
   - 选择"**其他**"

4. **网站语言**
   - 选择"**中文**"

5. **网站首页URL**
   - 填写：**https://geshixiugai.cn**
   - 或：**https://www.geshixiugai.cn**

6. **网站简介**
   - 填写：提供Word文档格式修改服务
   - 或：提供在线文档处理工具

7. **点击"下一步"**

### 2.3 上传资料

1. **下载网站真实性核验单**
   - 系统会自动生成核验单
   - 点击"**下载**"按钮下载PDF文件
   - 打印出来（A4纸）

2. **填写核验单**
   - 在打印的核验单上：
     - 填写网站名称：诚叔工具站
     - 填写域名：**geshixiugai.cn**
     - **手写签名**（必须手写，不能打印）
     - 填写日期

3. **拍照上传**
   - 用手机拍照（确保清晰）
   - 上传：
     - 身份证正面照片
     - 身份证反面照片
     - 网站真实性核验单照片（已签名）

4. **确认上传**
   - 检查照片是否清晰
   - 确认信息无误
   - 确认域名是 **geshixiugai.cn**（不是 .org）

5. **点击"提交审核"**

---

## 第三步：等待审核

### 3.1 审核流程

备案审核分为几个阶段：

1. **阿里云初审**（1-2个工作日）
   - 阿里云审核你的资料
   - 如果有问题，会通过短信或邮件通知你修改

2. **短信核验**（初审通过后）
   - 会收到工信部发送的验证码短信
   - 需要在24小时内完成核验
   - 访问：https://beian.miit.gov.cn/ 完成核验

3. **管局审核**（7-20个工作日）
   - 各省通信管理局审核
   - 这是最长的等待时间

### 3.2 查看审核进度

1. **登录备案系统**
   - 访问：https://beian.aliyun.com/
   - 登录账号

2. **查看备案进度**
   - 在"备案管理"页面可以看到：
     - 当前状态（审核中/已通过/已驳回）
     - 审核进度
     - 预计完成时间

### 3.3 处理审核问题

如果审核被驳回：

1. **查看驳回原因**
   - 在备案系统中查看驳回原因
   - 通常会有详细的说明

2. **修改信息**
   - 根据驳回原因修改信息
   - 重新上传资料

3. **重新提交**
   - 修改完成后重新提交

---

## 第四步：备案通过后

### 4.1 获取备案号

备案通过后，你会收到：
- **备案号**：如"浙ICP备12345678号-1"
- **备案密码**：用于后续备案管理

### 4.2 在网站添加备案号

**重要：必须在网站底部显示备案号！**

1. **获取备案号显示代码**
   ```html
   <a href="https://beian.miit.gov.cn/" target="_blank">备案号：浙ICP备12345678号-1</a>
   ```

2. **添加到网站底部**
   - 在网站底部（footer）添加备案号
   - 确保可以点击跳转到工信部备案查询网站

### 4.3 继续支付宝产品签约

备案完成后，回到支付宝开放平台：

1. **登录支付宝开放平台**
   - 访问：https://open.alipay.com/

2. **进入产品签约页面**
   - 应用详情 → 可调用产品 → 找到"电脑网站支付"

3. **填写备案信息**
   - 在签约时填写：
     - 备案号：浙ICP备12345678号-1
     - 网站地址：**https://geshixiugai.cn**（使用新域名）

4. **提交签约申请**
   - 确认信息无误后提交
   - 等待审核（几小时到1个工作日）

## 域名配置（备案完成后）

备案完成后，需要配置域名解析：

### 如果使用阿里云服务器

1. **在阿里云控制台配置 DNS**
   - 登录阿里云控制台
   - 进入"域名" → "解析设置"
   - 添加 A 记录指向服务器 IP

### 如果使用 Vercel

1. **在 Vercel 添加域名**
   - 登录 Vercel Dashboard
   - 进入项目 → Settings → Domains
   - 添加：geshixiugai.cn

2. **配置 DNS 记录**
   - 在域名注册商处配置 CNAME 记录
   - 指向 Vercel 提供的地址
