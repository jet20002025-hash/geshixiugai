# R2 权限选择确认

## ✅ 正确的选择

从你提供的选项列表中，选择：

**"Workers R2 存储"**（Workers R2 Storage）

---

## 📋 完整配置

### 三个下拉菜单应该选择：

1. **第一个下拉菜单**：`帐户`（Account）✅
2. **第二个下拉菜单**：`Workers R2 存储`（Workers R2 Storage）✅
3. **第三个下拉菜单**：`Edit`（编辑）✅

---

## 🎯 操作步骤

1. **点击第二个下拉菜单**
2. **在列表中查找并选择 "Workers R2 存储"**
3. **点击第三个下拉菜单**
4. **选择 "Edit"**（编辑权限）

---

## 📝 为什么选择 "Workers R2 存储"？

- ✅ 这是 R2 对象存储的权限
- ✅ 允许读写 R2 存储桶中的文件
- ✅ 符合我们的需求（存储文档文件）

**注意**：不要选择：
- ❌ "Workers R2 数据目录"（这是用于数据目录功能）
- ❌ "Workers R2 SQL"（这是用于 SQL 功能）

---

## ✅ 配置完成后

配置好权限后，继续：

1. **填写 Token 名称**：`word-formatter-r2-token`
2. **配置账号资源**：选择 "Include" → 选择你的账号
3. **配置域名资源**：选择 "None"
4. **点击 "Continue to summary"**
5. **检查配置，点击 "Create Token"**
6. **立即复制保存** Access Key ID 和 Secret Access Key

---

## 🎉 完成！

选择 "Workers R2 存储" 和 "Edit" 权限后，Token 就可以正常访问 R2 存储了！
