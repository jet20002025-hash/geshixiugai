# 从产品列表页面找到支付产品

## 当前情况

你现在在"**支付有礼**"产品页面，这是营销推广产品，不是支付产品。

## 需要找到的产品

对于"网页应用"，需要签约"**电脑网站支付**"产品（不是"支付有礼"）。

## 操作步骤

### 方法 1：从左侧导航栏查找

1. **查看左侧导航栏**
   - 当前选中的是"私域" → "支付有礼"
   - 需要找到"**公域**"或"**其他**"分类

2. **展开"公域"分类**
   - 点击左侧导航栏的"**公域**"（向下箭头表示可展开）
   - 在"公域"分类下，应该能找到"**电脑网站支付**"产品

3. **或者展开"其他"分类**
   - 点击左侧导航栏的"**其他**"
   - 支付产品可能在这个分类下

### 方法 2：使用搜索功能

1. **使用顶部搜索框**
   - 在页面右上角的搜索框输入："**电脑网站支付**"
   - 或搜索："**网页支付**"、"**page.pay**"

2. **查看搜索结果**
   - 点击搜索结果中的"电脑网站支付"产品

### 方法 3：返回应用详情页

1. **返回应用详情**
   - 点击页面顶部的"控制台"或应用名称
   - 或使用浏览器返回按钮，回到应用详情页面

2. **进入产品列表**
   - 在应用详情页面，找到"**产品列表**"或"**功能列表**"
   - 点击"**添加产品**"或"**签约产品**"按钮

3. **在产品选择页面查找**
   - 在产品选择页面，搜索"**电脑网站支付**"
   - 或在"公域"分类下查找

## 产品位置说明

支付产品通常在以下位置：

- **公域** 分类下：
  - 电脑网站支付（网页应用使用）
  - 手机网站支付（移动应用使用）
  - 当面付
  - 等等

- **其他** 分类下：
  - 某些支付产品也可能在这里

## 确认找到正确的产品

找到产品后，确认以下几点：

1. **产品名称**：电脑网站支付 ✅
2. **接口名称**：`alipay.trade.page.pay` ✅
3. **适用场景**：电脑网站支付 ✅

**注意**：不要选择"手机网站支付"（那是移动应用用的）

## 签约步骤

找到"电脑网站支付"产品后：

1. 点击"**立即签约**"或"**查看详情**"
2. 填写签约信息：
   - 产品使用场景：论文格式修复服务
   - 网站地址：https://geshixiugai.org
3. 提交签约申请
4. 等待审核（几小时到1个工作日）

## 如果找不到"电脑网站支付"

如果搜索和分类中都找不到"电脑网站支付"产品：

1. **确认应用类型**
   - 回到应用详情页面
   - 确认应用类型确实是"**网页应用**"

2. **联系支付宝客服**
   - 客服电话：95188
   - 说明：应用类型是"网页应用"，但在产品列表中找不到"电脑网站支付"产品

3. **检查是否满足条件**
   - 某些情况下，可能需要先完成企业认证
   - 或满足其他特定条件才能看到该产品
