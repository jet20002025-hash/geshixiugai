# 使用阿里云控制台连接服务器 - 最简单的方法

## 🎯 目标

连接到服务器 `121.199.49.1` 并执行部署命令。

---

## 📋 详细步骤

### 步骤 1：登录阿里云控制台

1. 访问：https://ecs.console.aliyun.com/
2. 使用你的阿里云账号登录

### 步骤 2：找到服务器实例

1. 登录后，点击左侧菜单"实例与镜像" → "实例"
2. 在实例列表中找到你的服务器
   - 可以通过公网 IP `121.199.49.1` 查找
   - 或通过实例名称查找

### 步骤 3：打开远程连接

1. 点击服务器名称或右侧的"管理"按钮
2. 在服务器详情页面，点击右上角的 **"远程连接"** 按钮
3. 选择连接方式：
   - **Workbench远程连接**（推荐 ⭐）
   - **VNC远程连接**（备用）

### 步骤 4：登录服务器

1. 在弹出的连接窗口中：
   - **用户名**：输入 `root`
   - **密码**：输入你设置的 root 密码
2. 点击"确定"或按回车

### 步骤 5：连接成功

连接成功后，你会看到一个终端界面，类似：

```
[root@iZxxxxx ~]#
```

现在可以执行命令了！

---

## 🚀 连接成功后执行部署

在远程连接的终端中，依次执行以下命令：

```bash
# 1. 创建项目目录
mkdir -p /var/www
cd /var/www

# 2. 克隆代码
git clone https://github.com/jet20002025-hash/geshixiugai.git geshixiugai
cd geshixiugai

# 3. 运行部署脚本
chmod +x deploy_aliyun.sh
sudo ./deploy_aliyun.sh
```

连接成功后，可以测试一下：

## 🔧 常见问题

### 问题 1：忘记 root 密码

**解决方法**：
1. 在服务器详情页面
2. 点击"更多" → "密码/密钥" → "重置实例密码"
3. 设置新密码
4. 重启服务器

### 问题 2：连接超时

**可能原因**：
- 网络问题
- 服务器未运行

**解决方法**：
1. 检查服务器状态（应该是"运行中"）
2. 尝试使用 VNC 远程连接
3. 检查网络连接

### 问题 3：提示"连接失败"

**解决方法**：
1. 刷新页面重试
2. 尝试使用 VNC 远程连接
3. 检查服务器是否正常运行
