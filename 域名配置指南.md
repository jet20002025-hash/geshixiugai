# 域名配置指南 - geshixiugai.org

## 域名信息
- **域名**: geshixiugai.org
- **注册商**: Spaceship
- **含义**: 格式修改（中文拼音）

## 配置方案

### 方案1: 使用 Cloudflare（推荐 ⭐）

#### 优势
- 免费 CDN 加速
- 免费 SSL 证书
- DDoS 防护
- 国内可访问
- DNS 管理简单

#### 配置步骤

**1. 在 Cloudflare 添加域名**

1. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com)
2. 点击 "Add a Site"
3. 输入域名：`geshixiugai.org`
4. 选择免费计划（Free Plan）
5. Cloudflare 会扫描现有 DNS 记录

**2. 在 Spaceship 修改 Nameservers**

1. 登录 Spaceship 账户
2. 找到域名 `geshixiugai.org`
3. 进入 DNS 管理
4. 修改 Nameservers 为 Cloudflare 提供的：
   ```
   [Cloudflare 会提供两个 nameservers，类似：]
   dante.ns.cloudflare.com
   lola.ns.cloudflare.com
   ```
5. 保存更改
6. 等待生效（通常几分钟到几小时）

**3. 在 Cloudflare 配置 DNS 记录**

在 Cloudflare DNS 设置中添加：

```
类型: CNAME
名称: @
目标: cname.vercel-dns.com
代理状态: 已代理（橙色云朵）

类型: CNAME
名称: www
目标: cname.vercel-dns.com
代理状态: 已代理（橙色云朵）
```

**4. 在 Vercel 添加自定义域名**

1. 登录 Vercel Dashboard
2. 进入你的项目
3. 点击 Settings → Domains
4. 点击 "Add Domain"
5. 输入：`geshixiugai.org`
6. 点击 "Add"
7. Vercel 会自动验证域名所有权

**5. 配置 SSL（自动）**

- Cloudflare 会自动提供 SSL 证书
- Vercel 也会自动配置 SSL
- 通常几分钟内生效

### 方案2: 直接在 Spaceship 配置（简单但不推荐）

#### 配置步骤

**1. 在 Vercel 获取 CNAME 记录**

1. 在 Vercel 项目设置中添加域名：`geshixiugai.org`
2. Vercel 会提供 DNS 配置信息

**2. 在 Spaceship 配置 DNS**

在 Spaceship DNS 管理中添加：

```
类型: CNAME
主机: @
值: cname.vercel-dns.com
TTL: 3600

类型: CNAME
主机: www
值: cname.vercel-dns.com
TTL: 3600
```

**注意**: 如果 Spaceship 不支持根域名 CNAME（@），需要：
- 使用 A 记录指向 Vercel IP（联系 Vercel 获取）
- 或使用 URL 重定向

## DNS 记录配置详解

### Cloudflare DNS 配置

| 类型 | 名称 | 内容 | 代理 | TTL |
|------|------|------|------|-----|
| CNAME | @ | cname.vercel-dns.com | ✅ 已代理 | Auto |
| CNAME | www | cname.vercel-dns.com | ✅ 已代理 | Auto |

### 说明
- **@**: 代表根域名（geshixiugai.org）
- **www**: 代表 www.geshixiugai.org
- **代理状态**: 开启（橙色云朵）会通过 Cloudflare CDN
- **TTL**: Auto（自动）

## 常见问题

### Q1: DNS 多久生效？
**A**: 通常 5 分钟到 24 小时，取决于：
- Nameservers 更改：几小时
- DNS 记录更改：几分钟到几小时

### Q2: 为什么访问不了？
**A**: 检查：
1. Nameservers 是否正确指向 Cloudflare
2. DNS 记录是否正确
3. Vercel 域名是否已添加
4. 等待 DNS 传播完成

### Q3: 如何强制刷新 DNS？
**A**: 
- 清除浏览器缓存
- 使用 `nslookup geshixiugai.org` 检查
- 等待 DNS 缓存过期

### Q4: 需要备案吗？
**A**: 
- 如果使用 Vercel（国外服务器）：**不需要备案**
- 如果使用阿里云（国内服务器）：**需要备案**
