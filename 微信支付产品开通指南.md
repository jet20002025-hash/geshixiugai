# 微信支付产品开通指南

## 📋 开通前准备

在开通支付产品之前，请确认：
- ✅ 微信支付商户号已审核通过（状态为"正常"）
- ✅ 已登录微信支付商户平台：https://pay.weixin.qq.com/

---

## 🚀 开通步骤

### 步骤 1：登录微信支付商户平台

1. 访问：https://pay.weixin.qq.com/
2. 使用微信扫码登录
3. 选择你的商户号（1732703522）

---

### 步骤 2：进入产品中心

1. 登录后，在顶部导航栏找到"**产品中心**"
2. 点击进入"产品中心"页面

**如果找不到"产品中心"**：
- 可能在左侧菜单栏
- 或者点击"更多"菜单查找

---

### 步骤 3：开通 H5 支付（网页支付）

**H5 支付**：用户在手机浏览器或PC浏览器中直接跳转到微信支付页面完成支付。

#### 3.1 找到 H5 支付产品

1. 在"产品中心"页面，找到"**H5支付**"或"**网页支付**"
2. 点击"**开通**"或"**申请开通**"按钮

#### 3.2 填写开通资料

需要填写以下信息：

1. **支付域名**
   - 填写你的网站域名：`geshixiugai.org`
   - 如果需要多个域名，可以添加：
     - `geshixiugai.org`
     - `www.geshixiugai.org`
   - **注意**：域名需要已备案（如果使用国内服务器）

2. **网站信息**
   - 网站名称：论文格式修复平台（或你的实际名称）
   - 网站描述：提供论文格式自动修复服务

3. **经营类目**
   - 选择"**教育/培训**"或"**其他服务**"
   - 根据你的实际业务选择

4. **上传资料**（如果需要）
   - 网站截图
   - 营业执照（个体工商户）
   - 其他相关证明文件

#### 3.3 提交审核

1. 检查填写的信息是否正确
2. 点击"**提交审核**"
3. 等待审核（通常 1-3 个工作日）

---

### 步骤 4：开通 Native 支付（扫码支付）

**Native 支付**：用户在PC端扫描二维码完成支付。

#### 4.1 找到 Native 支付产品

1. 在"产品中心"页面，找到"**Native支付**"或"**扫码支付**"
2. 点击"**开通**"或"**申请开通**"按钮

#### 4.2 填写开通资料

需要填写以下信息：

1. **支付场景**
   - 选择"**PC网站**"或"**其他场景**"
   - 描述：用户在PC端扫描二维码完成支付

2. **网站信息**
   - 网站名称：论文格式修复平台
   - 网站描述：提供论文格式自动修复服务

3. **经营类目**
   - 选择"**教育/培训**"或"**其他服务**"

4. **上传资料**（如果需要）
   - 网站截图
   - 营业执照

#### 4.3 提交审核

1. 检查填写的信息是否正确
2. 点击"**提交审核**"
3. 等待审核（通常 1-3 个工作日）

---

## ⏰ 审核时间

- **审核时间**：通常 1-3 个工作日
- **审核状态**：可以在"产品中心"查看审核进度
- **审核通过**：会收到微信通知，产品状态变为"已开通"

---

## ✅ 审核通过后

### 1. 确认产品状态

1. 进入"产品中心"
2. 确认以下产品状态为"**已开通**"：
   - ✅ H5支付
   - ✅ Native支付

### 2. 配置支付回调地址

1. 进入"**开发配置**"或"**API配置**"
2. 找到"**支付回调地址**"或"**支付通知URL**"
3. 设置回调地址：
   ```
   https://geshixiugai.org/payments/wechat/notify
   ```
   **重要**：
   - 没有 `/api` 前缀
   - 必须是 HTTPS
   - 必须是公网可访问的地址

4. 点击"**保存**"

### 3. 测试支付功能

1. 访问你的网站：https://geshixiugai.org/web/
2. 上传文档并处理
3. 选择"微信支付"
4. 点击"立即支付"
5. 测试支付流程

## ❓ 常见问题

### Q1: 找不到"产品中心"在哪里？

**A**: 
- 可能在顶部导航栏
- 或在左侧菜单栏
- 或点击"更多"菜单查找
- 如果还是找不到，可以联系微信支付客服

### Q2: 域名需要备案吗？

**A**: 
- 如果使用国内服务器，域名需要备案
- 如果使用海外服务器（如 Vercel），通常不需要备案
- 具体以微信支付的要求为准

### Q3: 审核被拒绝了怎么办？

**A**: 
1. 查看拒绝原因
2. 根据拒绝原因修改资料
3. 重新提交审核
4. 如果问题复杂，可以联系微信支付客服

### Q4: 可以只开通 H5 支付吗？

**A**: 
- 可以，如果只需要网页支付，只开通 H5 支付即可
- 如果需要在 PC 端使用，建议同时开通 Native 支付

### Q5: 开通需要多长时间？

**A**: 
- 通常 1-3 个工作日
- 如果资料齐全，可能更快
- 如果资料不完整，可能需要补充资料

### Q6: 开通需要费用吗？

**A**: 
- 开通产品本身不需要费用
- 但每笔交易会收取手续费（约 0.6%）
