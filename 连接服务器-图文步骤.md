# 连接服务器图文步骤 - 最简单的方法

## 🎯 目标

连接到服务器 `121.199.49.1` 并执行部署命令。

---

## 📱 方法 1：Mac 系统（最简单）

### 步骤 1：打开终端

1. 按 `Command + 空格键`
2. 输入"终端"或"Terminal"
3. 按回车

![Mac 终端](提示：你会看到一个黑色或白色的窗口)

### 步骤 2：输入连接命令

在终端中输入：
```bash
ssh root@121.199.49.1
```

然后按回车。

### 步骤 3：确认连接

如果是第一次连接，会提示：
```
The authenticity of host '121.199.49.1' can't be established.
Are you sure you want to continue connecting (yes/no)?
```

输入 `yes` 并回车。

### 步骤 4：输入密码

会提示：
```
root@121.199.49.1's password:
```

**注意**：输入密码时不会显示任何字符（包括 `*`），这是正常的！
- 直接输入你的 root 密码
- 输入完成后按回车

### 步骤 5：连接成功

如果成功，你会看到类似这样的提示符：
```
root@iZxxxxx:~#
```

现在就可以执行命令了！

---

## 💻 方法 2：Windows 系统

### 步骤 1：打开 PowerShell

**Windows 10/11**：
1. 按 `Win + X` 键
2. 选择"Windows PowerShell"或"终端"

**或者**：
1. 按 `Win + R` 键
2. 输入 `powershell`
3. 按回车

### 步骤 2：输入连接命令

在 PowerShell 中输入：
```powershell
ssh root@121.199.49.1
```

然后按回车。

### 步骤 3：确认和输入密码

- 如果是第一次连接，输入 `yes` 确认
- 输入 root 密码（不会显示）
- 按回车

### 步骤 4：连接成功

看到 `root@iZxxxxx:~#` 就表示连接成功了！

---

## 🖥️ 方法 3：使用 VS Code 或 Cursor（推荐）

### 为什么推荐？

- ✅ 可以在服务器上直接编辑文件
- ✅ 有代码高亮和自动补全
- ✅ 可以同时打开多个终端
- ✅ 可以浏览服务器文件系统

### 步骤 1：安装扩展

1. 打开 VS Code 或 Cursor
2. 点击左侧的扩展图标（四个方块）
3. 搜索 "Remote - SSH"
4. 点击"安装"

### 步骤 2：连接到服务器

1. 按 `F1` 键（或 `Ctrl + Shift + P`）
2. 输入 "Remote-SSH: Connect to Host"
3. 选择 "Add New SSH Host"
4. 输入：`ssh root@121.199.49.1`
5. 选择配置文件位置（默认即可）
6. 点击左下角的绿色图标
7. 选择 `121.199.49.1`
8. 输入密码

### 步骤 3：在服务器上操作

- 连接成功后，左侧文件浏览器会显示服务器文件
- 点击底部"终端"图标，打开服务器终端
- 现在可以执行命令了

## ✅ 连接成功后做什么？

看到 `root@iZxxxxx:~#` 后，执行以下命令：

```bash
# 1. 创建项目目录并克隆代码
mkdir -p /var/www && cd /var/www
git clone https://github.com/jet20002025-hash/geshixiugai.git geshixiugai
cd geshixiugai

# 2. 运行部署脚本
chmod +x deploy_aliyun.sh
sudo ./deploy_aliyun.sh
```

## 🔍 验证连接

连接成功后，可以测试一下：

```bash
# 查看当前目录
pwd

# 查看系统信息
uname -a

# 查看磁盘空间
df -h
```

如果这些命令都能正常执行，说明连接成功！

## ❌ 常见错误

### 错误 1：Connection refused

**原因**：服务器未开放 SSH 端口或服务未启动

**解决**：
1. 检查阿里云控制台 → 安全组 → 确保开放了 22 端口
2. 检查服务器是否正常运行

### 错误 2：Permission denied

**原因**：密码错误

**解决**：
- 重新输入正确的密码
- 如果忘记密码，可以在阿里云控制台重置

### 错误 3：ssh: command not found（Windows）

**原因**：Windows 版本太旧，没有内置 SSH

**解决**：
- 更新 Windows 到最新版本
- 或使用 PuTTY（下载：https://www.putty.org/）

## 💡 小技巧

### 1. 快速复制粘贴

- **Mac**：`Command + C` 复制，`Command + V` 粘贴
- **Windows**：`Ctrl + C` 复制，`Ctrl + V` 粘贴
- **注意**：在终端中，右键也可以粘贴

### 2. 断开连接

- 输入 `exit` 或按 `Ctrl + D`

### 3. 如果连接中断

- 重新执行 `ssh root@121.199.49.1` 即可
