## 三、Web版集成方案（推荐）

### 3.1 注册腾讯广告联盟媒体主账号

**重要**：您当前注册的是广告主账户（73845359），需要另外注册**媒体主账户**。

**⚠️ 重要提示**：如果系统平台下拉菜单只有"Android平台"、"iOS平台"、"Harmony平台"三个选项，说明您当前在**移动应用媒体注册页面**，不是网站媒体注册页面。

**正确的注册入口**：

1. **访问腾讯广告联盟（网站媒体专用）**：
   - 直接访问：https://union.qq.com/
   - 或搜索"腾讯广告联盟"（不是"腾讯优量汇"）
   - 选择"网站主"或"媒体主"注册

2. **如果只能访问 e.qq.com**：
   - 登录后，在后台顶部或左侧菜单寻找"媒体主"或"网站主"入口
   - 不要选择"应用媒体"或"移动应用"
   - 选择"网站媒体"或"Web媒体"

2. **注册媒体主账号**：
   - 使用QQ或微信登录（可以与广告主账户相同账号）
   - 选择"媒体主"或"网站主"角色
   - 填写基本信息
   - 上传资质证明（营业执照等）
   - 等待审核（通常1-3个工作日）

3. **创建媒体**：
   - 登录媒体主后台
   - 创建新的媒体（网站）
   - 填写网站信息：
     - 域名：geshixiugai.cn
     - 网站名称：论文格式自动修复
     - 网站类型：工具/服务类
     - ICP备案号：浙ICP备2025213303号-1
   - 提交审核

4. **创建广告位**：
   - 媒体审核通过后，创建广告位
   - 选择广告类型：横幅广告（推荐）
   - 设置广告位尺寸：728x90 或 300x250
   - 设置广告位位置：页面底部
   - **获取广告位ID（ad_id）** ← 这个才是需要的ID

### 3.2 获取广告代码

审核通过后，在广告位管理页面获取广告代码，格式类似：

```html
<!-- 腾讯广告联盟代码 -->
<script type="text/javascript" src="https://qzs.qq.com/qzone/biz/res/i.js" charset="utf-8"></script>
<script type="text/javascript">
  var AD_TAG = "你的广告位ID";
  var AD_WIDTH = 300;
  var AD_HEIGHT = 250;
</script>
<div id="ad_container"></div>
```

## 五、注意事项

### 1. 广告位位置选择
- **推荐位置**：页面底部、侧边栏
- **避免位置**：主要内容区域（影响用户体验）
- **移动端**：考虑响应式设计

### 2. 广告尺寸
- **PC端**：300x250、728x90、336x280 等
- **移动端**：320x50、300x250 等
- 根据页面布局选择合适的尺寸

### 3. 用户体验
- 不要遮挡主要内容
- 不要影响页面加载速度
- 确保广告与内容有明显区分

### 4. 收益优化
- 选择合适的广告位置
- 测试不同广告尺寸的效果
- 关注点击率（CTR）和收益数据

## 六、审核要求

### 需要准备的材料：
1. **营业执照**（企业/个体工商户）
2. **网站备案信息**（ICP备案号）
3. **网站内容**：确保内容合法合规
4. **网站访问量**：有一定访问量更容易通过审核

### 审核时间：
- 账号审核：1-3个工作日
- 媒体审核：1-3个工作日
- 广告位审核：通常即时通过
