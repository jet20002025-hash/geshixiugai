# 在哪里执行服务器命令

## 📍 命令执行位置说明

**所有服务器命令都在阿里云服务器上执行，不是在本地电脑！**

---

## 🖥️ 两种连接服务器的方式

### 方式 1：阿里云控制台网页终端（推荐新手）⭐

**优点**：
- ✅ 不需要安装任何软件
- ✅ 直接在浏览器中操作
- ✅ 适合不熟悉 SSH 的用户

**操作步骤**：

1. **登录阿里云控制台**
   - 访问：https://ecs.console.aliyun.com/
   - 使用你的阿里云账号登录

2. **找到你的服务器**
   - 在左侧菜单点击 **"实例与镜像"** → **"实例"**
   - 找到你的服务器（IP: 121.199.49.1）

3. **打开远程连接**
   - 点击服务器右侧的 **"远程连接"** 按钮
   - 选择 **"Workbench远程连接"** 或 **"VNC远程连接"**
   - 点击 **"立即登录"**

4. **输入密码**
   - 输入你的服务器 root 密码
   - 登录成功后，会显示命令行界面

5. **执行命令**
   - 在这个命令行界面中执行所有服务器命令
   - 例如：`cd /var/www/geshixiugai`、`git pull origin main` 等

**界面示例**：
```
[admin@iZbp1fic2d7eob78hrzn0hZ ~]$ 
```
这就是服务器命令行，在这里执行所有命令！

### 方式 2：SSH 连接（推荐有经验用户）

**优点**：
- ✅ 更快速
- ✅ 可以使用本地终端工具
- ✅ 支持文件传输

**操作步骤**：

#### Windows 用户

1. **使用 PowerShell 或 CMD**
   - 按 `Win + R`，输入 `powershell` 或 `cmd`
   - 打开命令行窗口

2. **连接服务器**
   ```bash
   ssh root@121.199.49.1
   ```

3. **输入密码**
   - 输入你的服务器 root 密码
   - 登录成功后，会显示服务器命令行

#### Mac 用户

1. **使用终端（Terminal）**
   - 按 `Command + 空格`，输入 `终端` 或 `Terminal`
   - 打开终端应用

2. **连接服务器**
   ```bash
   ssh root@121.199.49.1
   ```

3. **输入密码**
   - 输入你的服务器 root 密码
   - 登录成功后，会显示服务器命令行

## 🔍 如何区分命令在哪里执行

**特征**：
- 路径包含 `/Users/zwj/word格式修改器`
- 使用 `git push` 提交代码
- 修改本地文件

**示例**：
```bash

```

**执行位置**：
- Mac：终端（Terminal）

### 服务器命令（在阿里云服务器执行）

**特征**：
- 路径包含 `/var/www/geshixiugai`
- 使用 `git pull` 拉取代码
- 使用 `systemctl` 管理服务
- 需要先连接服务器

**示例**：
```bash

ssh root@121.199.49.1  # 先连接服务器
cd /var/www/geshixiugai
git pull origin main
sudo systemctl restart geshixiugai
```

**执行位置**：
- 方式 1：阿里云控制台网页终端
- 方式 2：SSH 连接后的终端

## ⚠️ 常见错误

### 错误 1：在本地执行服务器命令

**错误示例**：
```bash
# 在本地电脑执行
cd /var/www/geshixiugai  # ❌ 错误：本地没有这个路径
```

**正确做法**：
```bash
# 先连接服务器
ssh root@121.199.49.1

# 然后在服务器上执行
cd /var/www/geshixiugai  # ✅ 正确
```

### 错误 2：在服务器执行本地命令

**错误示例**：
```bash
# 在服务器上执行
cd /Users/zwj/word格式修改器  # ❌ 错误：服务器没有这个路径
```

**正确做法**：
```bash
# 在本地电脑执行
cd /Users/zwj/word格式修改器  # ✅ 正确
```
