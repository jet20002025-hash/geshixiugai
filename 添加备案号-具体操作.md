# 添加备案号 - 具体操作

## 🚀 操作步骤

### 步骤 1：编辑前端文件

在服务器上执行（阿里云控制台的对话框里）：

```bash
cd /var/www/geshixiugai
vi frontend/index.html
```

### 步骤 2：在 vi 编辑器中添加备案号

1. **找到页面底部**：
   - 按 `G` 跳转到文件末尾
   - 或使用方向键向下滚动到 `</body>` 标签附近

2. **进入编辑模式**：
   - 按 `i` 进入编辑模式

3. **添加备案号**：
   - 在 `</body>` 标签之前添加以下代码：

```html
<footer style="text-align: center; padding: 20px; color: #666; font-size: 12px; border-top: 1px solid #eee; margin-top: 40px;">
    <p>备案号：<a href="https://beian.miit.gov.cn/" target="_blank" style="color: #666; text-decoration: none;">浙ICP备2025213303号-1</a></p>
</footer>
```

4. **保存退出**：
   - 按 `Esc` 退出编辑模式
   - 输入 `:wq` 然后按回车保存退出

### 步骤 3：重启服务

```bash
# 重启服务使更改生效
sudo systemctl restart geshixiugai

# 检查服务状态
sudo systemctl status geshixiugai
```

### 步骤 4：验证备案号显示

在浏览器中访问：
- `https://geshixiugai.cn`
- 滚动到页面底部
- 应该看到备案号：`浙ICP备2025213303号-1`

## 💡 vi 编辑器使用提示

- `G` - 跳转到文件末尾
- `i` - 进入编辑模式
- `Esc` - 退出编辑模式
- `:wq` - 保存并退出
- `:q!` - 不保存退出
